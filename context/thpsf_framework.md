## องค์ประกอบที่ 1: ความรู้ (ร — Knowledge)

### ร 1.1 ความรู้ในศาสตร์สาขาวิชาของตน

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ร 1.1.1 | อธิบายแนวคิดและหลักการสำคัญของวิชาที่สอน พร้อมทั้งการประยุกต์ใช้ | 1 |
| ร 1.1.2 | จัดลำดับและเชื่อมโยงหัวข้อที่สอนได้อย่างเป็นระบบ | 1 |
| ร 1.1.3 | ติดตามองค์ความรู้ทันสมัยใน**ศาสตร์ของตน** วิเคราะห์และนำมาใช้อย่างเหมาะสม | 1–3 |
| ร 1.1.4 | ถ่ายทอดองค์ความรู้ให้แก่เพื่ออาจารย์ในสาขาวิชาหรือหน่วยงาน | 2 |
| ร 1.1.5 | ประเมินแนวคิด หลักการสำคัญในศาสตร์ของตน เพื่อเป็นแนวทางสร้างองค์ความรู้ใหม่ | 2 |
| ร 1.1.6 | เชื่อมโยงความรู้ใน**ศาสตร์ของตน**กับศาสตร์อื่นที่เกี่ยวข้อง | 2 |
| ร 1.1.7 | ให้คำปรึกษา ชี้แนะ ความรู้ในศาสตร์ของตนแก่อาจารย์ในองค์กร | 3 |
| ร 1.1.8 | สร้างองค์ความรู้ใหม่ของ**ศาสตร์ที่สอน** | 3 |
| ร 1.1.9 | คาดการณ์หรือประเมินทิศทางความก้าวหน้าของ**ศาสตร์ที่สอน** | 3 |

### ร 1.2 ความรู้ในศาสตร์การสอนและการเรียนรู้

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ร 1.2.1 | อธิบายจิตวิทยาการเรียนรู้และปัจจัยที่ส่งผลต่อการเรียนรู้ | 1 |
| ร 1.2.2 | ระบุปัจจัยชี้จัดการเรียนรู้และวิธีวัดและประเมินผลที่เหมาะสมกับผลการเรียนรู้และกลุ่มผู้เรียน | 1 |
| ร 1.2.3 | จัดลำดับและเชื่อมโยงหัวข้อที่สอนได้อย่างเป็นระบบ | 1 |
| ร 1.2.4 | แนะนำและแลกเปลี่ยนเรียนรู้วิธีการจัดการเรียนรู้และการวัดและประเมินผลแก่เพื่ออาจารย์ในสาขาวิชาหรือหน่วยงาน | 2 |
| ร 1.2.5 | ประเมินวิธีจัดการเรียนรู้ และการวัดและประเมินผล | 2–3 |
| ร 1.2.6 | บูรณาการการจัดการเรียนการสอนระหว่างวิชา หรือศาสตร์อื่นที่เกี่ยวข้อง | 1–2 |
| ร 1.2.7 | ถ่ายทอด หรือ ให้คำปรึกษา หรือ ชี้แนะวิธีการจัดการเรียนรู้และการวัดและประเมินผลแก่อาจารย์ในองค์กร | 3 |
| ร 1.2.8 | สร้างความรู้ หรือ นวัตกรรมใน**ศาสตร์การสอน** และการจัดการเรียนรู้ หรือให้คำปรึกษาระดับชาติ/นานาชาติ | 3–4 |

## องค์ประกอบที่ 2: สมรรถนะ (ส — Competencies)

### ส 2.1 ออกแบบและวางแผนการจัดกิจกรรมการเรียนรู้ของผู้เรียนอย่างมีประสิทธิภาพ

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ส 2.1.1 | ออกแบบกิจกรรม และการใช้เทคโนโลยีการเรียนรู้ที่สอดคล้องกับผลการเรียนรู้ (PLOs–Content–Teaching/Learning) | 1 |
| ส 2.1.2 | [Optional] เลือกวิธีและเครื่องมือในการวัดและประเมินผลที่สอดคล้องกับผลการเรียนรู้ และกำหนดเกณฑ์การประเมินที่ชัดเจน | 1 |
| ส 2.1.3 | ประเมินการออกแบบการจัดการเรียนรู้ และพัฒนาการจัดการเรียนรู้ | 2–3 |
| ส 2.1.4 | ประสานงานกับผู้สอนร่วมในรายวิชา (ถ้ามี) ในการออกแบบการเรียนรู้อย่างมีประสิทธิภาพ | 1 |
| ส 2.1.5 | มีส่วนร่วมในการพัฒนาการออกแบบการเรียนรู้ร่วมกับเพื่ออาจารย์ในสาขาวิชาหรือหน่วยงาน | 2 |

### ส 2.2 ดำเนินกิจกรรมการเรียนรู้ได้อย่างมีประสิทธิภาพ

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ส 2.2.1 | จัดการเรียนรู้ที่เน้นผู้เรียนเป็นศูนย์กลาง | 1 |
| ส 2.2.2 | เหมาะสมกับเนื้อหา ผลการเรียนรู้ และระดับของผู้เรียน | 1 |
| ส 2.2.3 | ใช้ทรัพยากรและสื่อการเรียนรู้ที่หลากหลายเพื่อสนับสนุนการเรียนรู้ | 1 |
| ส 2.2.4 | [Optional] ประเมินความก้าวหน้า (formative) ของผู้เรียนระหว่างดำเนินการสอน | 1 |
| ส 2.2.5 | [Optional] ประสานงานการจัดการเรียนรู้กับผู้สอนร่วม (ถ้ามี) อย่างมีประสิทธิภาพ | 1 |
| ส 2.2.6 | กำกับดูแลและติดตามพัฒนาการการเรียนรู้ของผู้เรียนอย่างเป็นระบบ | 1–2 |
| ส 2.2.7 | ใช้นวัตกรรมหรือความคิดสร้างสรรค์ในการจัดการเรียนรู้อย่างมีประสิทธิภาพ | 1–2 |
| ส 2.2.8 | แนะนำและแลกเปลี่ยนเรียนรู้วิธีจัดกิจกรรมการเรียนรู้แก่เพื่ออาจารย์ใน**ศาสตร์การสอน** | 2–3 |
| ส 2.2.9 | ทำวิจัยในชั้นเรียน (**ศาสตร์การสอน**) และนำผลมาพัฒนากิจกรรมการเรียนรู้ | 3 |

### ส 2.3 เสริมสร้างบรรยากาศการเรียนรู้และสนับสนุนการเรียนรู้ของผู้เรียน

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ส 2.3.1 | จัดบรรยากาศการเรียนรู้โดยการมีส่วนร่วมจากผู้เรียน | 1 |
| ส 2.3.2 | ให้คำปรึกษาแก่กลุ่มผู้เรียนในการพัฒนาการเรียนรู้ | 1–2 |
| ส 2.3.3 | จัดบรรยากาศการเรียนรู้ที่ส่งเสริมการเรียนรู้ร่วมกันระหว่างผู้เรียนกับผู้เรียน | 1 |
| ส 2.3.4 | วินิจฉัยข้อปัญหาของผู้เรียนและช่วยเหลืออย่างเหมาะสมทันเวลา | 1–2 |
| ส 2.3.5 | ประสานงานการจัดการเรียนรู้กับผู้สอนร่วม (ถ้ามี) อย่างมีประสิทธิภาพ | 1 |
| ส 2.3.6 | ริเริ่มการเปลี่ยนแปลงบรรยากาศการเรียนรู้ในระดับหลักสูตร (**ศาสตร์การสอน**) | 3 |

### ส 2.4 วัดและประเมินผลการเรียนรู้ของผู้เรียน พร้อมทั้งสามารถให้ข้อมูลป้อนกลับอย่างสร้างสรรค์

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ส 2.4.1 | ประเมินความก้าวหน้า (formative) และประเมินผลสรุป (summative) โดยเน้นผลสัมฤทธิ์การเรียนรู้ | 1 |
| ส 2.4.2 | สามารถให้ข้อมูลป้อนกลับแก่ผู้เรียนอย่างสร้างสรรค์ | 1 |
| ส 2.4.3 | [Optional] เลือกวิธีและเครื่องมือในการวัดและประเมินผลที่สอดคล้องกับผลการเรียนรู้ | 1 |
| ส 2.4.4 | วัดและประเมินผลโดยใช้วิธีการใหม่ๆ เพื่อพัฒนาประสิทธิภาพและประสิทธิผลของการเรียนรู้ | 2–3 |
| ส 2.4.5 | ประเมินวิธีการวัดและประเมินผลเพื่อนำมาพัฒนาการเรียนรู้ของผู้เรียน | 2–3 |

## องค์ประกอบที่ 3: ค่านิยม (ค — Values)

### ค 3.1 คุณค่าในการพัฒนาวิชาชีพอาจารย์ และการพัฒนาตนเองอย่างต่อเนื่อง

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ค 3.1.1 | พัฒนาตนเองอย่างต่อเนื่อง | 1 |
| ค 3.1.2 | มีส่วนร่วมในการส่งเสริมให้เกิดการพัฒนาวิชาชีพอาจารย์ในสาขาวิชาหรือหน่วยงาน | 2–3 |
| ค 3.1.3 | รับฟังความคิดเห็นจากผู้ร่วมวิชาชีพเพื่อนำมาพัฒนาตนเอง | 1–3 |
| ค 3.1.4 | เป็นพี่เลี้ยงและผู้นำในการพัฒนาวิชาชีพอาจารย์ในองค์กร | 3–4 |

### ค 3.2 ธำรงไว้ซึ่งจรรยาบรรณแห่งวิชาชีพอาจารย์

| รหัส | รายละเอียดสมรรถนะ | ระดับ |
|------|-------------------|-------|
| ค 3.2.1 | ปฏิบัติตามจรรยาบรรณแห่งวิชาชีพอาจารย์ขององค์กร | 1 |
| ค 3.2.2 | มีส่วนร่วมในการส่งเสริมให้เกิดความเข้าใจและปฏิบัติตามจรรยาบรรณฯ ในสาขาวิชาหรือหน่วยงาน | 2 |
| ค 3.2.3 | เป็นผู้พี่เลี้ยง และผู้นำเชิงนโยบายด้านจรรยาบรรณแห่งวิชาชีพอาจารย์ในองค์กร | 3 |
